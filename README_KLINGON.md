## Configuration

- `image`: nagh mIllogh.
- `cols`: columns.
- `rows`: rows.
- `scale_factor`: mIllogh tInqu'choH.
- `border_color`: cher ghu' color.
- `border_width`: cher ghu' width.
- `piece_bg_color`: nagh Dop color.
- `scene_bg_color`: mIw ghu' color.
